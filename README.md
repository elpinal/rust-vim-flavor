# Rust Vim Flavor

`Rust-vim-flavor` is a Rust port of [`vim-flavor`](https://github.com/kana/vim-flavor).

## Install

In this repository, use [`Cargo`](http://doc.crates.io/guide.html):

```bash
$ cargo install
```

then, an executable, named 'vim-flavor', will be installed.
